=== Stata -b (batch mode) exit behavior ===
- Batch mode always returns exit code 0 to the shell, even on fatal errors.
- Error propagation is internal (r-class return codes), not via process exit code.

=== Documented r-class codes (from real Stata run) ===
rc=0:   Success (no error)
rc=1:   Return code 1 (generic)
rc=9:   Assertion failed (assert 1==0)
rc=111: Invalid syntax
rc=198: Invalid use of Mata function
rc=601: File not found

=== Key behaviors ===
- `stata-se -b do file.do` → exit 0 for ALL outcomes (success or error)
- `stata-se -e 'command'` → exit 0 for success; also exit 0 for error (batch mode)
- Uncaught errors print: "r(CODE);" and terminate the do-file
- Caught errors (via capture): set _rc but do NOT print r(CODE);
- Error messages are SMCL-free in text log mode
