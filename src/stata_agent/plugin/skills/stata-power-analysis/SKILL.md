# Power Analysis

Use this skill when precision and detectability are the focus.

1. Clarify the estimand and target effect size.
2. State the assumptions behind the power calculation.
3. Distinguish formal power analysis from ex post rationalization.

Use `stata run` to execute power commands (e.g., `power twomeans`, `power regress`, `power sampsi`) and `stata results` to retrieve computed power, sample size, or effect size values.

Read `references/power-checklist.md` for the reporting checklist.
