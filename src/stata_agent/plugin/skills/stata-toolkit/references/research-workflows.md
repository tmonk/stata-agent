# Research Workflows

Use the toolkit as the entrypoint for these paper-oriented workflows:

- Replication and robustness checks.
- Data audit before estimation.
- Publication QA for tables and graphs.
- Referee-response re-runs with specification comparisons.
- Causal inference workflows where identifying assumptions and diagnostics matter.
- Environment diagnosis on university-managed or locked-down machines.

Prefer the specialized skills whenever a workflow has repeated judgment points or multiple execution steps.
