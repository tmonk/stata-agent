# Table Patterns

Review:

- column comparability,
- sample size drift,
- coefficient labels,
- notes for controls, FE, clustering, and weights,
- consistent rounding.

Treat a table as reader-facing prose in numeric form, not just extracted estimates.

Use `stata results --return e` to extract authoritative stored results and `stata run` to produce comparison tables via `esttab`, `tabout`, or `table`.
