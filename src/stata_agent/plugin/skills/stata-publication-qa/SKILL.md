# Publication QA

Use this skill when editorial quality matters.

1. Re-run or inspect the relevant model or graph.
2. Review statistical clarity, labeling, sample consistency, and visual presentation.
3. Distinguish required fixes from optional polish.

Use `stata run` to re-run models, `stata results` to verify stored e() results, `stata inspect describe` to check sample consistency, and `stata graph list`/`stata graph export` to review visual outputs.

Read `references/checklist.md` for the review checklist.
